> 关联：阶段一/二/三（金库分发 / 拉高出货 / 聪明钱vs笨蛋钱）。
> 性质：复盘提炼，非投资建议；信号需后续多样本回测才能从"一个故事"升级为"有胜率的方法"。

## 三、把它变成"可执行的东西"——几个方向（待选）

1. **信号 Checklist / 评分卡**：把 S1~S8 做成一张"任何新 TGE 代币 0-100 分风险/出货评分卡"，输入合约地址即可打分。
2. **多样本回测验证**：拿 2~3 个其他"解禁即砸"代币（同类低流通 TGE）跑同一套信号，验证**普适性与胜率**——这是把"一个故事"升级为"方法"的关键一步。
3. **实时监控/告警 Dashboard**：盯一组目标代币的"金库→EOA 发弹药 + 净 CEX 流入 + DEX 刷量比 + 解锁日历"，异动告警。
4. **可传播图谱**：把 WCT 三部曲做成桑基资金流 + 价格出货叠加的叙事图，沉淀方法论。

## 四、诚实边界（决定 α 的可靠度）
- **CEX 黑箱**：真实成交、接盘散户、内部人 CEX 利润都看不到 → 信号是"出货准备"的代理，非成交本身；实时性受限。
- **选择偏差**：不是每个 TGE 都这样。WCT 是一个样本，**必须回测多样本**才能谈胜率，否则是事后归因。
- **博弈会进化**：一旦信号公开，操盘方会规避（换更隐蔽的备货路径）。α 会衰减。
- 非投资建议。

## 六、心法：找到"不会骗人的数据源"（可迁移的元认知）

> 这是整个项目最该带走的一条。

**任何市场,先找到它"不会骗人的底层数据源",再用纪律去读它。**
- 币圈 = **链上数据**;股市 = **财报/招股书/内部人申报(Form 4)/机构持仓(13-F)**;期货 = 持仓/库存报告……
- **工具不同,功夫一样**:看证据、不看叙事;跟资金/内部人,不跟喊单。

### 链上 vs 财报——类比的"妙"与"断"
**妙(链上某些方面更猛)**
- 财报:季度、滞后、经会计修饰;链上:**实时、逐笔、结算层无法造假**。
- 股市内部人卖股要事后申报;币圈内部人砸盘**你几乎同步看到**(如 40M 巨鲸进币安)。
- 财报是"公司说的版本",链上资金流是"**他们做了什么的铁证**"。

**断(三处类比会失效,本项目全踩到)**
1. **链上 ≠ 全貌**:CEX 订单簿成交、做市合同、OTC、VC 条款、真实身份——全看不到(⛔)。链上对结算层完整,对链下那一半是瞎的 → **实时但残缺**。
2. **多数币没"基本面"**:没有现金流/利润锚。所谓基本面其实是**代币经济学/供应/筹码流向** → 这是**仓位/资金流的游戏,不是价值的游戏**。更像读"股东名册+内部人减持+解禁表",而非读利润表。
3. **监管天差地别**:拉高出货、内部人解禁砸盘,在股市多属违法会被查;在币圈**是公开的标准操作,没人管**——这正是 WCT 剧本能明目张胆上演的原因。

### 最该记住的一句
**知道"该看哪个数据"远远不够。** 多数人就算把财报/链上摆面前还是亏——因为 **"会读数据 + 情绪纪律"才是真稀缺**。数据源是必要条件,不是充分条件。
> 本项目全程坚持的"**要证据、分清 ✅能证/🔶推断/⛔不可知、不自欺**",才是真正能跨市场复用的内功。
