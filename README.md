# WCT-analyse

一次**链上取证级**的代币全生命周期调查：从合约部署、10 亿铸造，到 2025 年 5 月的拉高出货，再到一片死寂（−95%）——全程用链上数据，像开了上帝视角一样还原 WalletConnect Token（WCT）的**代币流向、解锁机制、巨鲸/受益人、操盘手法与赢家输家**，并从中提炼可复用的 α 方法论。

> ⚠️ **声明**：本仓库是**研究 / 教育**性质的链上数据分析，**不构成任何投资建议**。所有结论严格区分 **✅实证 / 🔶推断 / ⛔不可知**；涉及"操纵"只描述**链上行为与时间巧合**，不作法律定性。

调查对象：WCT（OP Mainnet 代理合约 `0xeF4461891DfB3AC8572cCf7C794664A8DD927945`）。
数据源：链上（Dune Analytics）+ 官方文档 + 区块浏览器 + 合约源码。

## 心法（最该带走的一条）

> **任何市场，先找到它"不会骗人的底层数据源"，再用纪律去读它。** 币圈是链上，股市是财报/Form 4/13-F——工具不同，功夫一样。数据源是必要不充分，**会读数据 + 情绪纪律**才是真稀缺。

## 诚实边界

链上能看到资金流、持仓、解锁、充提；**看不到 CEX 订单簿成交、真实买卖价、链下身份与做市合同**。因此一切美元金额为**估算/下限**，"接盘散户"主体在 CEX、链上不可见。报告中以 ✅/🔶/⛔ 标注每条结论的证据强度。
